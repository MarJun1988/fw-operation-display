# Mitteilungsicon

In diesem Bereich werden alle **Icons** angezeigt, die aktuell von der Anwendung bereitgestellt werden.

Die Icons werden in einer **&uuml;bersichtlichen Liste mit Vorschau** dargestellt und k&ouml;nnen zur visuellen
Kennzeichnung von
Mitteilungen verwendet werden.
