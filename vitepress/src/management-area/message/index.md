# Mitteilungen

Der Men&uuml;punkt **Mitteilungen** dient dazu, Einsatzkr&auml;fte &uuml;ber allgemeine Hinweise und Informationen zu
informieren.  
Dazu z&auml;hlen beispielsweise Hinweise zu **Straßensperrungen**, **Einbahnstrassen**, **Baustellen** oder sonstige
relevante
Informationen im Einsatzgebiet.

Die erstellten Mitteilungen k&ouml;nnen – sofern entsprechend konfiguriert – **automatisch im Dashboard angezeigt**
werden.

&Uuml;ber die **Sortierung** der Eintr&auml;ge wird festgelegt, in welcher **Reihenfolge** die Mitteilungen im Dashboard
dargestellt werden.
