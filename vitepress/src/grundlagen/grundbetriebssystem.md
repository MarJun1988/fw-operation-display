# Information zum Grundbetriebssystem

Die Web-Anwendung kann grunds&auml;tzlich auf **jedem System betrieben werden**, das die Nutzung von **Docker**
unterst&uuml;tzt.

Empfohlen wird der Einsatz eines **UNIX-basierten Betriebssystems**, da Docker hier besonders stabil und performant
l&auml;uft.  
Dazu z&auml;hlen unter anderem:

- **Linux**
- **macOS**

F&uuml;r den Betrieb auf einem **Raspberry Pi** wird in der Regel eine Linux-Distribution verwendet, die speziell auf
die **ARM-Prozessorarchitektur** abgestimmt ist.

Diese Kombination hat sich im praktischen Einsatz als zuverl&auml;ssig und ressourcenschonend erwiesen und eignet sich
besonders gut f&uuml;r den Dauerbetrieb im Feuerwehrhaus.
